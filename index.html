Custom Shortcut Setup

\ \

This page will allow you to create a homescreen shortcut with a custom icon.

\ \
\

To add to your homescreen:

\
    \
  1. In Chrome, tap the three-dot menu ( \f1 \uc0\u8942 \f0 ) in the top-right corner
    2. \
  2. Select "Add to Home screen"
    3. \
  3. Confirm by tapping "Add"
    4. \
\

Once added, the shortcut will open your desired page with the custom icon.
